# Quick Newsletter Tab Setup

## Create Newsletter Tab in Google Sheets (30 seconds)

1. **Open your Google Sheet:**
   https://docs.google.com/spreadsheets/d/13LQj17MTo0gPB3dGKR9P8qu0XTVvpCTpF04pUZak6U4/edit

2. **Click the "+" button** at the bottom left (next to "Entries" tab)

3. **Name the new sheet:** `Newsletter`

4. **Add headers in row 1:**
   - Column A: `Email`
   - Column B: `Discount Code`
   - Column C: `Timestamp`

5. **Format header row (optional but nice):**
   - Select row 1
   - Make it bold
   - Add gray background color

**That's it!** The newsletter emails will automatically start saving to this tab.

---

## What Gets Saved

When someone signs up via the popup:
- **Email:** Their email address
- **Discount Code:** Unique 5% off code (e.g., `THEOS5ABC123`)
- **Timestamp:** When they signed up

---

## Test It

1. Visit your homepage
2. Wait 2 seconds for popup
3. Enter a test email
4. Check the Newsletter tab - you should see the entry!
